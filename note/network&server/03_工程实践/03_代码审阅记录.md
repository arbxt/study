## 2. 当前已确认的长期审阅重点

```text
1. fd 生命周期是否清晰：谁创建、谁监听、谁关闭。
2. I/O 等待是否和业务处理混在一起。
3. worker 是否错误地 read/close fd。
4. 是否正确处理 read/recv 返回值：>0、==0、<0。
5. select/poll/epoll 的事件集合是否与实际 fd 状态一致。
6. 是否存在 fd 被关闭后又被任务队列中的旧任务使用的问题。
7. 是否有短写、EINTR、EAGAIN/EWOULDBLOCK 等错误处理缺口。
8. 是否已经到了需要 Connection/read_buffer/write_buffer 的阶段。
```
